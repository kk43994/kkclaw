FEATURES核心特性

Everything an AI Companion Needs

Built from scratch with pure code animations, zero sprite sheets.

Air-Feel UI

空气感 UI

67px fluid glass ball with 7 mood-based color themes. No clunky windows — just a floating orb on your desktop.

67px 流体玻璃球，7 种情绪色系。没有笨重窗口，只有一个漂浮的光球。

15+ Eye Expressions

15+ 眼睛表情

Capsule eyes blink, look around, and react. Idle micro-animations trigger randomly — your pet is always alive.

胶囊眼睛会眨眼、环顾、做出反应。待机微表情随机触发 — 宠物始终鲜活。

Emotional TTS

情感语音

MiniMax Speech 2.5 Turbo with voice cloning + 7 emotion modes. 3-tier fallback: MiniMax → CosyVoice → Edge TTS.

MiniMax Speech 2.5 Turbo 声音克隆 + 7 种情绪。三级降级：MiniMax → CosyVoice → Edge TTS。

Desktop Lyrics

桌面歌词

Agent responses float like desktop lyrics — typewriter effect, auto-fade, fully click-through. Never blocks your work.

AI 回复像桌面歌词一样浮现 — 打字机效果、自动淡出、完全鼠标穿透。

Dual Window Architecture

双窗口架构

Sprite window (200×220, draggable) + Lyrics window (400×100, click-through). Both transparent, always-on-top.

精灵窗口 (200×220, 可拖动) + 歌词窗口 (400×100, 鼠标穿透)。双透明、双置顶。

Pure Code Animation

纯代码动画

All animations procedurally generated with CSS + JS. Zero sprite sheets, zero image assets. Everything is code.

所有动画均由 CSS + JS 程序化生成。零精灵图、零图片资源。一切都是代码。
