# 60s Demo 视频分镜草案（desktop-pet）

目标：60 秒内展示桌面龙虾作为 OpenClaw 可视化与运维外壳的核心价值。

## 时间线
1. 0-6s：启动画面（Logo + 一句话定位）
2. 6-18s：球体待机与情绪变化（idle/happy/thinking）
3. 18-30s：接收消息并语音播报（桌面歌词窗同步）
4. 30-42s：Gateway 异常检测 → 自动恢复（guardian）
5. 42-52s：模型切换/状态面板（连接、请求、错误）
6. 52-60s：收尾 CTA（GitHub / Release / 下载）

## 拍摄素材清单
- 启动到在线全过程录屏（10-15s）
- 情绪球体特写（至少 3 种状态）
- 一次真实消息播报过程（含歌词窗）
- 一次故障恢复演示（可用重启网关模拟）
- 诊断面板数据截图（连接、错误、健康度）

## 输出规格
- 主视频：1920x1080, 60s, mp4
- 预览动图：最长 10-15s，控制体积用于 README
